## Library Project

A simple library project to add and remove books from library. You can also toggle Book's read status.

### Limitations

- No persistent storage. All data gets lost on page reload.

### Things learned

- Polished my layout skills
- Learned about the concept of backgrop and modal
- How to associate DOM elements with objects using **data attributes**.
- Write good css with custom properties
- Strict discipline of following preset conditions for better organization. (Gets more rewarding when codebase grows)
- Learned about z-index property and how to center fixed elements (fixed positioning) using **transform: translate(-50%, -50%)**
- JS prototypical inheritance
